🔵 Steam Integration
Steam has a public API. Enter your Steam ID and a free API key to automatically import your entire Steam library. Find your Steam ID · Get a free API key
🔴 GOG Galaxy — Auto Import
Nexus reads your GOG Galaxy database directly — no export needed. Just make sure GOG Galaxy is installed and you've logged in at least once so your library has synced locally.

First time: you'll need to install one extra package. Run this once in your terminal from the nexus-library folder:
npm install better-sqlite3
🟡 Epic & 🟠 Amazon — Auto Import via Heroic
Since you have Heroic Games Launcher installed, Nexus reads both your Epic and Amazon Games libraries in one click. Make sure you're logged into both accounts in Heroic and your libraries have synced.
Fallback — Manual CSV Import
If Heroic import fails, you can manually create a CSV instead.
🟩 Xbox Personal Library & PC Game Pass
Connect your Xbox account via OpenXBL — a free, community-built Xbox Live API — to import your personal Xbox game library and browse the full PC Game Pass catalog.

Step 1: Create a free account at xbl.io, sign in with your Xbox account, and copy your personal API key from your profile page.

⚡ Two separate imports:
Xbox Library — your personally played games, from achievement history. Goes into your main library as owned Xbox titles.
Game Pass Catalog — the full list of games currently on PC Game Pass. These are not owned — they're only visible when you filter by "Game Pass" in the sidebar, and never counted in your library totals.
PERSONAL LIBRARY
Games you've played — added to your library as owned Xbox titles
GAME PASS CATALOG (not owned)
Catalog of available GP titles — isolated from your owned games
🎨 Cover Art — IGDB
Steam games automatically get cover art from Steam's CDN with no setup needed. For GOG, Epic, and manually added games, Nexus uses IGDB (Internet Game Database).

Get your free credentials at dev.twitch.tv/console → Register Your Application → Manage → copy your Client ID and generate a Client Secret.

💰 Price Tracking — gg.deals
gg.deals tracks prices across 40+ stores including Steam, GOG, Epic, Humble, Fanatical, Green Man Gaming, and key resellers like Kinguin and G2A. Get your free API key at gg.deals/api — log in, go to your profile settings, and generate a key.

Nexus downloads the full Steam game catalogue so you can search by name and resolve App IDs for any game, even ones not in your library. Uses your Steam API key from the import settings above.
🎮 Game Database — RAWG
RAWG has 500,000+ games across all platforms — perfect for enriching GOG, Epic, and Amazon titles with descriptions, genres, screenshots, and Metacritic scores. Get your free API key at rawg.io/apidocs — sign up and your key appears instantly on the page.
